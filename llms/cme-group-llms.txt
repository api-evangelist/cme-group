# CME Group

> CME Group is the world's largest derivatives exchange (CME, CBOT, NYMEX, COMEX), exposing cloud REST and WebSocket APIs for reference data, market data, benchmarks, and analytics. All cloud APIs are secured with OAuth 2.0 client-credentials (Bearer tokens from auth.cmegroup.com); access is entitlement-based per registered API ID. No public OpenAPI is published — message specifications live on the CME Group Client Systems Wiki.

Generated by API Evangelist (2026-07-22, method: generated) from apis.yml and repo artifacts.

## APIs

- [CME Reference Data API v3](https://www.cmegroup.com/trading/market-tech-and-data-services/cme-reference-data-api.html): Product and instrument reference data for CME Group, BrokerTec, EBS, and cleared markets. Base: https://refdata.api.cmegroup.com/refdata/v3 (endpoints: /products, /instruments, /tradingSchedules, /displayGroups, /tradingSessionValueDates, /optionSeries)
- [CME FedWatch End-of-Day API](https://www.cmegroup.com/markets/interest-rates/cme-fedwatch-tool.html): FOMC rate-change probabilities from 30-Day Fed Funds futures. Base: https://markets.api.cmegroup.com/fedwatch/v1 (endpoints: /forecasts, /meetings/future, /meetings/history)
- [CME FedWatch Intraday API](https://cmegroupclientsite.atlassian.net/wiki/spaces/EPICSANDBOX/pages/1054343170): 60-second-frequency FOMC probabilities. Base: https://markets.api.cmegroup.com/fedwatch_rt/v1
- [CME Term SOFR API](https://www.cmegroup.com/market-data/cme-group-benchmark-administration/term-sofr.html): 1M/3M/6M/12M Term SOFR reference rates. Base: https://markets.api.cmegroup.com/term-sofr-rates/v1 (endpoints: /latest, /history)
- [Greeks and Implied Volatility REST API](https://www.cmegroup.com/market-data/market-data-api.html): Five-minute snapshot Greeks and implied volatility for top 40 liquid products. Base: https://markets.api.cmegroup.com/greeks/v1
- [EUR/USD Cross Currency Basis Index API](https://www.cmegroup.com/market-data/market-data-api.html): Cross-currency basis index values. Base: https://markets.api.cmegroup.com/xccy/v1 (endpoints: /latest, /history)
- [CME ClearPort API](https://www.cmegroup.com/clearport/clearport-api.html): FIXML-based OTC trade submission for clearing.
- [Real-Time Futures and Options Data API](https://www.cmegroup.com/market-data/real-time-futures-and-options-data-api.html): Real-time prices via REST and WebSocket subscription.
